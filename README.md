# Library-Management-Database
Demo database project for internship

This project demonstrates  a relational database design for a simple library management system.
I am creating this to showcase understanding of database schema design, relationships, and SQL queries

# Its entites include-
1. Members
2. Books
3. Authors
4. Category(of Books)
5. Borrow_records

# It's features are -
1.Primary and foreign key constraints
2. JOIN-based queries for practical use
3. Normalized relational schema
4. Insertion of sample data

# Database: 
SQL (MySQL/PostgreSQL compatible)
